# Pi-Studio

Pi-Studio is a desktop application for managing and interacting with Pi coding agents. It provides a modern, feature-rich interface for session management, skill development, configuration, and provider integration.

## Features

- **Modern UI**: Built with Electron and React, featuring a clean, intuitive interface.
- **Session Management**: Create, manage, and monitor multiple Pi agent sessions.
- **Skill Development**: Tools for developing and managing agent skills.
- **Configuration Management**: Centralized management for agent configurations.
- **Provider Integration**: Seamless integration with various Pi providers.
- **Terminal Integration**: Built-in terminal for running commands within sessions.
- **File Management**: Open files and folders directly from the application.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Pi-Studio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

To run the application in development mode:

```bash
npm run dev
```

### Build

To build the application for production:

```bash
npm run build
```
